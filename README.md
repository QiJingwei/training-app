# 训练计划 PWA

一个专为个人训练设计的轻量 PWA，支持力量训练引导和跑步数据记录。可添加到手机主屏幕，用起来和 App 一样。

## 功能

**训练 Tab**
- 自动识别今天是周几，显示当日计划
- 支持点击周条切换查看其他日期
- 力量训练按轮次循环引导（方案一：5个动作依次完成第1组→第2组→第3组）
- 计时动作自动倒计时，次数动作手动确认
- 组间自动休息倒计时，最后3秒变红提示
- 训练完成后显示用时和完成组数

**跑步 Tab**
- 填入距离和时长，配速自动计算
- 记录平均心率、跑步类型、备注
- 历史记录列表，支持删除
- 保存成功有提示

**趋势 Tab**
- 汇总总公里数、总次数、平均配速、平均心率
- 距离 / 配速 / 心率三条折线图趋势

## 训练计划

按本人当前课表内置：

| 星期 | 内容 |
|------|------|
| 周一 | 休息 |
| 周二 | 跑步（Z2 常规 45min）|
| 周三 | 力量 A（臀腿 30min）|
| 周四 | 跑步（Z2 常规 45min）|
| 周五 | 力量 B（核心 25min）|
| 周六 | 长跑（Z2 长距 65min）|
| 周日 | 休息 |

## 添加到手机桌面

1. 手机 Safari 打开部署链接
2. 底部点分享按钮
3. 选「添加到主屏幕」

## 技术栈

纯 HTML / CSS / JS，无框架，无构建工具。数据存储在浏览器 localStorage。部署在 Netlify。

## 本地运行

直接用浏览器打开 `index.html` 即可。
